Current Projects

Automotive RFID Door

See full project details here.

My goal for this project was to design and install an RFID reader to activate my car door actuators. The biggest design challenge has been making something that would be able to read the signal at a tolerable distance while also being placed inside the car. This project is still in progress and im currently troubleshooting the relays and actuator wiring.

Lefty Keyboard

See full project details here.

I've built a few keyboards from individual components and decided it was time to try a custom PCB creation. I prefer full sized (109 keys) keyboards, which are much harder to find barebone PCBs for. I'm implementing a "lefty" feature where the numpad is to the left of the alphabetical keys. I'll be using KiCad for this project.

Game Pad

See full project details here.

I experienced a unique issue while using mouse and keyboard for gaming, in that longer nails dont fit in the empty space between keycaps. To remedy this, I used a 16 switch board and custom 3D printed caps to allow proper spacing. This is a completed project.
